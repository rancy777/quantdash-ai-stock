# Data Policy

公开仓库默认只保留目录骨架、少量非敏感示例文件，以及运行所需的最小说明。

以下内容默认不应进入公开仓库：

- 大体积 K 线历史库
- 单日导出快照和个人研究快照
- 第三方资讯抓取结果
- 自动抓取的研报原文、网页快照和本地缓存
- 本地认证数据库、会话状态、个人观察清单
- 个人整理的高手持仓或研究资料

如果你克隆的是公开仓库：

1. 先按 [README.md](../README.md) 安装依赖。
2. 再通过 `npm run sync:*` 或 Python 采集脚本自行生成本地数据。
3. 不要把第三方来源的数据、原文 PDF、网页快照或个人研究文件直接提交回仓库。
